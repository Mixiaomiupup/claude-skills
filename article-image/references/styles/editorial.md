# editorial

Magazine-style infographic, tech explainer.

## Color Palette

| Role | Color | Hex |
|------|-------|-----|
| Primary | Deep Indigo | #3C366B |
| Secondary | Coral | #FC8181 |
| Background | Light Gray | #F7FAFC |
| Accent | Teal | #38B2AC |
| Text | Dark Slate | #1A202C |

## Visual Elements

- Bold typography, data visualizations
- Structured sections, clear callouts
- Professional layout, magazine-quality composition

## Style Rules

### Do

- Strong hierarchy, bold headlines
- Data-driven visuals, clear sections

### Don't

- Casual or hand-drawn feel, playful elements
- Irregular layouts

## Best For

Tech explainers, journalism, data stories, long-form analysis.
