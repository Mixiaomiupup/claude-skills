# blueprint

Precise technical drawing with engineering aesthetics.

## Color Palette

| Role | Color | Hex |
|------|-------|-----|
| Background | Blueprint Paper | #FAF8F5 |
| Grid | Light Gray | #E5E5E5 |
| Primary Accent | Engineering Blue | #2563EB |
| Secondary | Navy Blue | #1E3A5F |
| Tertiary | Light Blue | #BFDBFE |
| Warning | Amber | #F59E0B |
| Text | Deep Slate | #334155 |

## Visual Elements

- Precise lines with consistent stroke weights
- Technical schematics, 90-degree connections
- Dimension lines, isometric projections, minimal charts

## Style Rules

### Do

- Consistent line weights, grid alignment
- Restrained palette, geometric precision

### Don't

- Hand-drawn shapes, decorative flourishes
- Curved connections, photographic elements

## Best For

System design, architecture, data analysis, infrastructure, process flows.
