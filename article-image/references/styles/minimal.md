# minimal

Ultra-clean, zen-like, essential.

## Color Palette

| Role | Color | Hex |
|------|-------|-----|
| Background | Pure White | #FFFFFF |
| Primary | Near Black | #1A202C |
| Accent | Single color (context-dependent) | varies |

## Visual Elements

- Single focal point, generous whitespace (60%+)
- Essential shapes only, thin precise lines

## Style Rules

### Do

- Maximum negative space, single concept focus
- Extreme simplicity

### Don't

- Add decoration, use multiple colors
- Fill empty space, add detail

## Best For

Philosophy, core concepts, zen, essential ideas.
